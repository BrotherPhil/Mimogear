A Static Site Generator & Web Development Framework

Mimogear was built from the ground up as a static site generator & a skeleton framework for pug:html, postcss:css, and javascript. By mixing in Gulp, popular task runner, Mimogear will automatically compile, distribute, and watch development assets.

  2. Fork
  3. Star
  4. Docs

Give Mimogear a try with a few setup commands in your terminal:

$ git clone https://github.com/mimoduo/Mimogear.git
$ cd Mimogear
$ npm install
$ npm install gulp-cli -g
$ gulp
