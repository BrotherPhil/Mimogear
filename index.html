Mimogear

Robust to the Core

Mimogear was built from scratch using modern front end practices. Under the hood you'll find gulp, pug, postcss, and javascript. My goal was to create a static site generator that accomplishes a variety of interface challenges while keeping it as simple as possible. I hope you enjoy how easy it is to build as much as I enjoyed creating it!
