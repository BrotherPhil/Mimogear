A Simple Static Site Generator

Mimogear was built from the ground up to serve as a skeleton framework for html/pug, css/postcss, and javascript. By integrating gulp, a popular task runner, Mimogear will automatically compile, distribute, and watch development assets. I hope you enjoy how easy it is to build as much as I enjoyed creating it!

Give Mimogear a try with a few setup commands in your terminal:

$ git clone https://github.com/mimoduo/Mimogear.git
$ cd Mimogear
$ npm install
$ gulp

When it comes to pug, a popular html templating language, Mimogear has you covered. With pug's low barrier of entry, you'll be pumping out content pages in no time flat. There are also some starter partials available in Mimogear to help style general interface elements at the beginning. If you'd like to learn more about pug before diving in, be sure to check our their awesome set of documentation. Just don't forget to come back!
You'll find postcss, a post-processor for css, has both the nature of plain css and the power of popular tools like sass. As an extendible language and a set of Mimogear defaults, you may never look back to other languages. Mimogear also provides a bare skeleton to help you get set up quickly so you can focus on the important elements of your design.
Javascript is available on every browser and has tons and tons of documentation. It's a fairly easy language to begin with but if you're feeling a little woozy on this one don't worry; Mimogear has you covered with a default set of modules.
